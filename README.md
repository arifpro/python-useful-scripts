# python-useful-scripts

## Use setup_env.sh to set up your virtual environment

- Make the script executable:

```bash
chmod +x setup_env.sh
```

- Run the script to set up your virtual environment and install dependencies:

```bash
./setup_env.sh
```

## Format code
